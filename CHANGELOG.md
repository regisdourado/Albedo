# Changelog

## [1.0.0] - 2026-02-07

### Adicionado
- Implementação inicial do AlbedoMaps
- Dashboard interativo com visualização de mapas de calor
- Integração com API Google Gemini para análise inteligente
- Componentes de metodologia e documentação
- Análise de dados Landsat 8/9 para LST (Land Surface Temperature)
- Cálculo de NDVI (Índice de Vegetação Normalizado)
- Integração com NASA POWER API para dados climáticos
- Seção sobre aspectos socioambientais
- Plano estratégico de arborização urbana
- Sistema de navegação por seções
- Interface responsiva com TailwindCSS
- Suporte a múltiplas visualizações (Hero, Dashboard, Methodology, About)
- Licença MIT
- README completo em Português
- Guia de contribuição (CONTRIBUTING.md)
- Código de Conduta (CODE_OF_CONDUCT.md)
- Template de configuração (.env.local.example)

### Tecnologias
- React 18.2
- TypeScript 5.2
- Vite 5.0
- TailwindCSS 3.4
- Recharts 2.10
- Lucide Icons
- Google Gemini API 1.40
